Myntra Clone
A front-end clone of the Myntra homepage, built using plain HTML5 and CSS3. This project recreates the core layout of Myntra's landing page, including the navigation bar, search bar, banner, category sections, and footer.

🚀 Features


Responsive navigation bar with logo, category links (Men, Women, Kids, Home & Living, Beauty, Studio), and a "New" badge
Search bar with a Material Symbols search icon
Action bar with Person, Wishlist, and Bag icons (Font Awesome)
Hero banner section
"Medal Worthy To Brands" offer grid with 12 promotional images
"Shop By Cart" category grid with 10 product images
Footer with multiple link columns and copyright notice


🛠️ Built With


HTML5 – Page structure and semantic markup
CSS3 – Styling and layout (Flexbox)
Font Awesome 6.7.2 – Icons (user, heart, bag)
Google Material Symbols – Search icon
